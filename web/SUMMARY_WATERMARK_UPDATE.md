# Summary Duration + Watermark Update

## 변경 내용

- 마지막 요약 카드에서 `최종 정리` 제목을 제거했습니다.
- 마지막 장점/단점/한줄평 장면의 표시 시간을 8초에서 10초로 소폭 연장했습니다.
- 모든 장면과 마지막 요약 장면에 `TuneMyFi` 반투명 워터마크를 추가했습니다.
  - 중앙에 아주 연한 브랜드 워터마크
  - 우하단에 식별용 워터마크

## 효과

- 마지막 정리 화면이 더 깔끔해집니다.
- 장점/단점 텍스트를 읽을 시간이 조금 더 확보됩니다.
- 다른 사람이 그대로 가져다 쓰기 어렵도록 브랜드 식별 표시가 영상 전반에 남습니다.
