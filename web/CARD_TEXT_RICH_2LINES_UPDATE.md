# Card Text Rich Two-Line Update

## 변경 내용

- 장면별 텍스트 카드 문장을 너무 짧은 키워드형이 아니라 자연스러운 2줄 문장으로 생성하도록 프롬프트를 조정했습니다.
- 각 줄은 28~42자 정도의 완성형 문장으로 요청합니다.
- 화면 카드에서는 숫자를 계속 아라비아 숫자로 표시합니다.
- TTS 발음용 숫자 변환 로직은 기존처럼 Spoken Script에만 적용됩니다.
- 긴 2줄 문장이 잘 들어가도록 카드 텍스트 폭을 약간 넓히고, 본문 폰트 크기와 카드 높이 기준을 조정했습니다.
