# Narration Positive-Balance Update

## 변경 내용

나레이션 생성 톤을 조정했습니다.

- 단점은 숨기지 않고 반드시 언급합니다.
- 다만 전체 흐름은 제품의 장점과 매력이 더 잘 드러나도록 구성합니다.
- 장점은 구체적인 사용 장면과 함께 먼저 설명합니다.
- 단점은 구매 전 확인 포인트처럼 차분하게 정리합니다.
- 장점/단점 비중은 대략 65:35 정도로 유도합니다.
- 마지막 요약은 장점 3개, 단점 2개, 긍정적 추천 톤의 한줄평으로 유도합니다.

## 사용 방법

기존에 생성된 나레이션에는 자동 반영되지 않습니다.
새 톤을 적용하려면 `/workflows`에서 `Prepare Narration Script`를 다시 실행하세요.
