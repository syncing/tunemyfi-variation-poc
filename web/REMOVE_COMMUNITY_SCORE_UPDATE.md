# Remove Community Score Update

## 변경 내용

- 롱폼 리뷰 나레이션 프롬프트에서 커뮤니티 평점/별점/점수 표현 금지 규칙을 추가했습니다.
- 쇼츠 나레이션 프롬프트에서도 커뮤니티 평점/별점/점수 표현 금지 규칙을 추가했습니다.
- 화면 카드/마지막 summary 생성 프롬프트에서도 평점/별점/점수 표현을 금지했습니다.
- Final Verdict JSON 스키마에서 community_score 생성을 제거했습니다.
- 기본 카드 문구의 `커뮤니티 리뷰` 표현을 `실제 사용자 의견`으로 바꿨습니다.

## 결과

롱폼/쇼츠 모두 장점, 단점, 한줄평 중심으로만 구성되고 커뮤니티 평점/점수 항목은 생성하지 않습니다.
